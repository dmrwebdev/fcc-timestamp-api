Microservice UNIX/UTC Timestamp API

Example Usage:

Type the date in either Unix time or Year-Month-Day at the end of the URL for a JSON timestamp!

  • /api/timestamper/1992-07-08
  • /api/timestamper710553600000

Example Output:

{"unix":710553600000, "utc":"Wed, 08 Jul 1992 00:00:00 GMT"}
